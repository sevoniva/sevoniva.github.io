flowchart TB
  A["确认问题现象"] --> B["确认影响范围"]

## 提交问题时建议提供

| 信息 | 示例 |
| --- | --- |
| 环境 | 单机、离线、Kubernetes、外部 MySQL |
| 版本 | Crest 版本号、部署时间 |
| 用户 | 发生问题的账号和角色 |
| 页面 | 具体菜单、资源名称、访问地址 |
| 时间 | 问题发生时间，精确到分钟更好 |
| 现象 | 页面报错、空白、无权限、导出失败等 |
| 日志 | 应用日志、审计日志、浏览器错误 |
