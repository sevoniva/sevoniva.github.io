---
title: 数据门户
description: Crest 数据门户使用说明，覆盖普通用户登录后的资源查看、收藏、最近使用、搜索筛选、预览和权限边界。
---

数据门户是普通用户查看已发布资源的入口。它面向“查看数据、筛选数据、全屏展示或下载结果”的使用场景，不要求用户进入后台资源树，也不要求用户具备编辑权限。

v1.5.5 起，Crest 提供独立门户能力。普通用户登录后会进入门户，在授权范围内查看已发布的仪表盘和数据大屏；具备后台权限的管理员、数据分析人员和资源维护人员，仍可进入后台工作台进行创建、编辑和管理。

## 门户适合哪些场景

| 场景 | 说明 |
| --- | --- |
| 经营看板查看 | 部门负责人每天查看经营指标、项目进度或风险数据 |
| 项目验收 | 项目组成员查看已发布的验收仪表盘或数据大屏 |
| 会议室展示 | 使用已授权账号打开大屏，进入全屏展示 |
| 普通用户只读访问 | 用户只查看资源，不进入后台编辑器 |
| 分级授权交付 | 管理员按用户、组织或角色授权不同资源 |

门户不是公开分享链接。门户访问仍要求用户登录，并且只展示当前账号被授权的资源。需要给未登录人员临时访问时，应使用分享功能，并按分享安全要求设置密码、有效期和访问范围。

## 进入门户

在浏览器中打开 Crest 访问地址，完成登录后进入系统。普通用户通常会进入数据门户；如果当前账号具备后台访问权限，可从页面入口或账号菜单进入后台工作台。

门户页面主要包含以下区域：

| 区域 | 用途 | 常见操作 |
| --- | --- | --- |
| 我的收藏 | 固定常用资源 | 收藏、取消收藏、打开资源 |
| 最近使用 | 展示最近打开过的资源 | 快速回到上一次查看的页面 |
| 全部资源 | 展示当前账号可访问的已发布资源 | 按名称、类型、更新时间查找 |
| 搜索框 | 按关键字查询资源 | 输入资源名称、业务关键词 |
| 类型筛选 | 区分仪表盘和数据大屏 | 只看某一类资源 |
| 账号菜单 | 当前用户相关操作 | 退出登录、进入后台或修改账号信息 |

## 查看资源

门户中展示的资源必须同时满足两个条件：

1. 资源已经发布。
2. 当前账号具备查看权限。

打开资源后，门户查看页会提供返回、刷新、全屏和账号菜单。数据大屏会按发布后的展示效果打开；仪表盘会按已发布版本展示。普通用户在门户中不会看到后台资源树的复制、移动、重命名、删除等管理入口。

如果页面中有查询组件，可按页面设计筛选数据。筛选结果只影响当前查看，不会修改资源本身。

## 收藏和最近使用

收藏用于固定高频资源，例如“经营日报”“项目交付看板”“值班监控大屏”。收藏后，资源会出现在“我的收藏”中。

最近使用由系统自动记录。用户打开已授权的仪表盘或数据大屏后，该资源会进入最近使用列表，便于下次快速返回。

资源被删除、取消发布或回收权限后，收藏和最近使用中的入口可能失效。遇到这种情况，应先在全部资源中搜索名称；仍找不到时，联系管理员检查资源状态和授权范围。

## 搜索和筛选

资源较多时，建议先使用关键字搜索，再配合资源类型筛选。

| 查找目标 | 建议操作 |
| --- | --- |
| 知道资源名称 | 直接输入名称关键字 |
| 只看大屏 | 类型选择“数据大屏” |
| 只看分析页面 | 类型选择“仪表盘” |
| 最近看过但忘记名称 | 打开“最近使用” |
| 高频资源 | 收藏后从“我的收藏”进入 |

资源名称应由维护人员提前规范。例如“经营驾驶舱”“研发交付周报”“项目风险监控”比“测试大屏”“新建仪表盘”更适合给普通用户使用。

## 全屏、刷新和返回

门户查看页常用操作：

| 操作 | 适用场景 |
| --- | --- |
| 返回 | 回到门户列表，继续打开其他资源 |
| 刷新 | 数据已更新或图表加载异常时重新请求 |
| 全屏 | 会议室、展厅或值班屏展示 |
| 账号菜单 | 退出登录或进入后台入口 |

大屏全屏展示前，建议先确认浏览器缩放比例为 100%，并检查目标屏幕分辨率。字体、图片、表格和图表边缘应在全屏后重新看一遍。

## 为什么看不到资源

门户只展示当前账号可访问的已发布资源。看不到资源时，按以下顺序排查：

1. 资源是否已经发布。
2. 资源类型是否被当前筛选条件隐藏。
3. 资源名称是否输入错误。
4. 当前用户、所在组织或角色是否拥有该资源查看权限。
5. 资源是否被移动、删除或取消发布。
6. 是否使用了错误账号登录。

普通用户不需要自己排查后台配置。可把资源名称、当前账号、截图和发生时间发给管理员，由管理员检查资源状态、角色权限和审计日志。

## 门户与后台的区别

| 项目 | 数据门户 | 后台工作台 |
| --- | --- | --- |
| 主要对象 | 普通查看用户 | 管理员、数据分析人员、资源维护人员 |
| 资源范围 | 已发布且已授权的资源 | 具备菜单和资源权限的可管理资源 |
| 常见操作 | 查看、收藏、搜索、刷新、全屏 | 新建、编辑、发布、授权、导出、系统管理 |
| 管理入口 | 不展示复制、移动、重命名、删除等后台操作 | 根据权限展示管理操作 |
| 安全边界 | 登录态 + 资源查看权限 | 菜单权限 + 功能权限 + 资源权限 |

仅查看页面时，应优先使用数据门户；只有需要创建数据源、维护数据集、编辑仪表盘或管理大屏时，才进入后台工作台。
