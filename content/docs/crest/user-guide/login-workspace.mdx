登录和工作台是所有操作的起点。工作台用于查看当前账号可访问的资源、进入常用页面、快速创建资源，以及回到最近访问过的内容。

## 打开登录页

在浏览器中输入管理员提供的 Crest 访问地址。

![登录页](/images/docs/crest/screenshots/login.png)

如果页面打不开，先确认：

| 检查项 | 说明 |
| --- | --- |
| 地址是否正确 | 域名、端口、协议必须和管理员提供的一致 |
| 网络是否可达 | 内网系统可能需要 VPN 或办公网络 |
| 浏览器是否支持 | 建议使用 Chrome、Edge 或企业统一推荐浏览器 |
| 是否被代理拦截 | 外部网络访问内网地址可能会失败 |

## 本地账号登录

<Steps>
  <Step>
    ### 输入账号
    在账号输入框填写管理员分配的用户名、邮箱或账号标识。
  </Step>
  <Step>
    ### 输入密码
    填写密码。首次登录通常使用管理员提供的初始密码。
  </Step>
  <Step>
    ### 点击登录
    登录成功后进入工作台。登录失败时先确认大小写、空格和输入法状态。
  </Step>
  <Step>
    ### 首次修改密码
    使用初始密码登录后，请立即进入账号菜单修改密码。
  </Step>
</Steps>

## 单点登录

如果企业已启用 SSO，登录页会显示单点登录入口。

操作流程：

1. 点击“单点登录”。
2. 浏览器跳转到企业统一认证页面。
3. 使用企业账号完成认证。
4. 认证成功后自动返回 Crest。
5. 首次登录后，系统会根据管理员配置创建或匹配用户。

如果 SSO 登录后没有菜单或资源，多数情况下是默认角色或资源权限未配置。请联系管理员检查用户角色。

## 工作台区域说明

登录成功后进入工作台。

![工作台](/images/docs/crest/screenshots/workspace.png)

| 区域 | 用途 | 常见操作 |
| --- | --- | --- |
| 顶部菜单 | 切换主要功能模块 | 工作台、仪表盘、数据大屏、数据血缘、数据准备 |
| 账号菜单 | 当前用户相关操作 | 修改密码、系统设置、退出登录 |
| 资源统计 | 当前账号可见资源数量 | 快速判断自己是否已有资源权限 |
| 快速创建 | 快速进入新建流程 | 新建仪表盘、大屏、数据集、数据源 |
| 我的收藏 | 展示收藏资源 | 进入高频访问页面 |
| 最近使用 | 展示最近访问资源 | 回到刚才编辑或查看的页面 |

## 快速创建入口

工作台上的快捷入口适合从零开始创建资源。

| 快捷入口 | 进入页面 | 使用前要准备什么 |
| --- | --- | --- |
| 仪表盘 | 仪表盘编辑器 | 至少有一个可用数据集 |
| 数据大屏 | 大屏编辑器 | 明确大屏主题、分辨率和数据集 |
| 数据集 | 数据集编辑器 | 至少有一个数据源 |
| 数据源 | 数据源管理 | 数据库、Excel 或 API 信息 |

首次使用 Crest 时，推荐顺序是“数据源 → 数据集 → 仪表盘或数据大屏”。直接新建仪表盘但没有数据集，会在图表配置阶段受阻。

## 收藏资源

收藏用于把高频访问资源固定到工作台。

典型场景：

- 每天早上查看的经营日报。
- 自己负责维护的部门仪表盘。
- 正在交付的大屏项目。
- 经常需要检查的数据集或分析页面。

资源被删除、移动或取消授权后，收藏入口可能失效。遇到这种情况，先在资源列表中搜索资源名称，再联系资源负责人确认。

## 最近使用

最近使用记录会显示当前账号最近打开过的资源。它适合快速返回上一项工作。

如果最近使用为空，可能原因包括：

| 原因 | 处理 |
| --- | --- |
| 当前账号第一次登录 | 正常现象，先打开几个资源后再查看 |
| 访问记录被清理 | 重新访问资源即可生成记录 |
| 资源被删除 | 在资源列表或审计日志中确认 |
| 资源权限被回收 | 联系管理员检查权限 |

## 修改密码

入口：右上角账号菜单 → 修改密码。

![修改密码](/images/docs/crest/screenshots/modify-password.png)

操作步骤：

<Steps>
  <Step>
    ### 打开修改密码
    点击右上角账号名称，在菜单中选择“修改密码”。
  </Step>
  <Step>
    ### 填写原密码
    输入当前正在使用的密码。
  </Step>
  <Step>
    ### 填写新密码
    输入符合管理员要求的新密码，并在确认密码中再次输入。
  </Step>
  <Step>
    ### 保存
    提交后重新登录，确认新密码生效。
  </Step>
</Steps>

SSO 用户的密码通常由企业身份平台管理。如果修改密码入口不可用，请使用企业账号体系修改。

## 常见问题

| 问题 | 可能原因 | 处理 |
| --- | --- | --- |
| 登录提示账号或密码错误 | 密码错误、账号不存在、输入法带入空格 | 重新输入，必要时联系管理员重置 |
| 登录成功但没有菜单 | 角色未授权菜单 | 联系管理员检查角色权限 |
| 工作台资源数量为 0 | 没有资源权限或尚未创建资源 | 先由管理员授权或创建资源 |
| 快速创建按钮不可用 | 当前账号缺少创建权限 | 申请数据源、数据集、仪表盘或大屏创建权限 |
| SSO 登录后用户信息不对 | 字段映射不正确 | 联系管理员检查 SSO 用户字段 |
| 修改密码失败 | 原密码错误或新密码不符合规则 | 检查密码规则后重试 |
